# GamerStake MVP

This is a beginner MVP for your gaming betting platform idea.

## Features
- Create game bets
- Display active bets
- Simple frontend UI

## Free Hosting Platforms
1. Vercel
2. Netlify
3. Firebase Hosting

## How to Launch on Vercel
1. Create a GitHub account
2. Upload these files to a GitHub repository
3. Go to vercel.com
4. Import your GitHub repo
5. Click Deploy
